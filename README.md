# python-getpaid-przelewy24

Przelewy24 payment gateway integration for the
[python-getpaid](https://github.com/django-getpaid) ecosystem.

> **Disclaimer:** This project has nothing in common with the Plone `getpaid`
> plugin. It is part of the `django-getpaid` / `python-getpaid` ecosystem.

## Installation

```bash
pip install python-getpaid-przelewy24
```

## Requirements

- Python 3.12+
- `python-getpaid-core >= 0.1.0`
- `httpx >= 0.27.0`
